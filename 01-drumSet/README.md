# Javascript 30 Challenge

## 1st challenge of #javascript30 - a drum set

## How to Run

Clone the directory and open the `index.html` file in your browser- no localhost server needed!

## How to Play

The keys for the drumset are:

V - Kick drum

B - 808 boom

N - Tom

F - Hi-hat

G - Open-hat

H - Snare

T - Clap

Y - Ride

U - Tink
